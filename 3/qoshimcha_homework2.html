CSS Qiyinlashtirilgan Vazifa

1-bo‘lim (Qizil)

bu birinchi matn bloki. u qizil circle bilan bog‘langan. margin, padding va border ishlatilgan.

2-bo‘lim (Ko‘k)

bu ikkinchi matn bloki. circle rangi o‘zgartirilgan. css class qo‘shish orqali rang boshqarilgan.

3-bo‘lim (Yashil)

bu uchinchi blok. yashil circle va yashil matn ishlatilgan. class orqali stil ajratilgan.

4-bo‘lim (Qora)

bu oxirgi blok. qora circle ishlatilgan. css class kombinatsiyasi amalda ko‘rsatilgan.
