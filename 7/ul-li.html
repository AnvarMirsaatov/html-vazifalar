CSS O'rganamiz (Katta Amaliyot)

CSS Xususiyatlari

▪ Rang: color - matn rangini belgilaydi.
▪ Orqa fon: background-color - fon rangini belgilaydi.
▪ Hajm: width, height - element o'lchamini boshqaradi.
▪ Matn tekislash: text-align - matnni joylashtiradi.
▪ Chegara: border - element atrofiga chegara qo'shadi.
▪ Yumaloqlik: border-radius - burchaklarni yumaloqlashtiradi.
▪ Ro'yxat: list-style - ro'yxat belgilarini boshqaradi.
▪ Animatsiya: transition, transform.
▪ Flexbox: display:flex - moslashuvchan joylashuv yaratadi.
▪ Grid: display:grid - katakli joylashuv yaratadi.
▪ Tashqi masofa: margin - elementlar orasidagi masofa.
▪ Ichki masofa: padding - element ichidagi bo'sh joy.
▪ Soya: box-shadow - elementga soya beradi.
▪ Matn soyasi: text-shadow - matnga soya beradi.
▪ Shaffoflik: opacity - ko'rinish darajasini o'zgartiradi.
▪ Harf o'lchami: font-size - matn hajmini o'zgartiradi.
▪ Qalinlik: font-weight - matn qalinligini belgilaydi.
▪ Ko'rinmas qilish: display:none - elementni yashiradi.
▪ Joylashuv: position - element pozitsiyasini boshqaradi.
▪ Qatlam: z-index - elementlar ustma-ustligini boshqaradi.
