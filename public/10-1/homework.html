📦 CSS BOX MODEL MAVZUSI

📚 Box Model Nima?

HTML dagi har bir element quti kabi ishlaydi. Bu qutining width, border, padding va margin qismlari bo‘ladi.

CSS yordamida biz shu qutining rangini, kattaligini va tashqi ko‘rinishini boshqaramiz.

🧠 Padding Nima?

Padding element ichidagi bo‘sh joy hisoblanadi. U text bilan border orasidagi masofani ochadi.

Padding katta bo‘lsa, text borderga yopishib qolmaydi.

📏 Margin Nima?

Margin elementning tashqarisidagi bo‘sh joydir.

Margin yordamida elementlar bir-biridan uzoqlashadi.

Tepadan katta margin berilgani uchun bu blok yuqoridagi blokdan uzoqlashdi.

🟩 Border Nima?

Border elementning chetiga chiziq chizadi.

Borderning qalinligi, rangi va turi bo‘ladi.

Bu blokda qalin yashil border ishlatilgan.

📐 Width Nima?

Width elementning kengligini belgilaydi.

Width kichik bo‘lsa, blok torroq ko‘rinadi.

⭕ Border Radius Nima?

Border-radius element burchaklarini yumaloq qiladi.

Radius katta bo‘lsa, burchaklar yanada yumaloq bo‘ladi.

🎉 Siz CSS Box Model mavzusining asosiy qismlarini o‘rgandingiz!

✅ Padding — ichki bo‘sh joy
✅ Margin — tashqi bo‘sh joy
✅ Border — chegara chizig‘i
✅ Width — kenglik
✅ Border Radius — yumaloq burchak
